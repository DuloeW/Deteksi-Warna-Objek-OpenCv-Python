# Clone this repo
`https://github.com/DuloeW/Deteksi-Warna-Objek-OpenCv-Python.git`

# Instalation
## Install the package
```bash
pip install opencv-python
pip install imutils
pip install numpy
```
## Active the venv
```bash
.venv\Scripts\Activate
```

## Run the file
```bash
python detect_color.py
```
